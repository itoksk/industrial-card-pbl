# Industrial Card PBL

TCG（トレーディングカードゲーム）制作を通じて資格知識を学ぶPBL教材セットです。

## ソース情報

- 独立リポジトリ: https://github.com/itoksk/industrial-card-pbl
- ライセンス: MIT

## 公開中URL

- 生徒向けサイト: https://otu4-mbna.vercel.app/
- 教員向けサイト: https://otu4-zv8p.vercel.app/
- カード生成ツール: https://otu4-5c8j.vercel.app/

## ディレクトリ構成

- `student-site/`: 生徒向け授業サイト
- `teacher-guide/`: 教員向けガイドサイト
- `card-generator/`: カード画像生成ツール

## Vercel設定（Root Directory）

- Student Site: `student-site`
- Teacher Guide: `teacher-guide`
- Card Generator: `card-generator`
