# ユーザーマニュアル

危険物カード画像生成ツールの使い方ガイドです。

---

## はじめに

このツールは、乙4危険物取扱者試験の学習に使える「危険物カード」の画像をAIで生成します。カードゲームのような見た目で、楽しく危険物の知識を学べます。**カードの枠・文字・レイアウトは固定テンプレートで統一**され、AIは**中央イラストのみ**を生成します。

---

## 準備するもの

- **APIキー**: 先生からもらったGoogle AI StudioのAPIキー
- **ブラウザ**: Chrome、Safari、Firefoxなど（最新版推奨）

---

## 基本的な使い方

### ステップ1: APIキーを入力

1. ページ上部の「APIキーを入力」欄に、先生からもらったAPIキーを貼り付けます
2. 「接続済み」と表示されればOK

> **注意**: APIキーは他の人に教えないでください

### ステップ2: 入力方法を選ぶ

7つのタブから好きな方法を選べます：

| タブ | 用途 | おすすめ度 |
|------|------|-----------|
| 📚 テンプレート | 用意されたサンプルから選ぶ | ★★★ 初心者向け |
| 🧪 物質情報 | 物質のデータを入力して生成 | ★★☆ 中級者向け |
| 🧯 消火方法 | 消火カードを作る | ★★☆ 中級者向け |
| 🚚 運搬 | 運搬カードを作る | ★★☆ 中級者向け |
| 📜 法令 | 法令カードを作る | ★★☆ 中級者向け |
| 📦 一括発行 | CSVから複数カードを発行 | ★★☆ 中級者向け |
| ✏️ 直接入力 | 自分でプロンプトを書く | ★☆☆ 上級者向け |

### ステップ3: 画像を生成

1. 情報を入力したら「📝 プロンプトを生成」をクリック（**中央イラスト用の指示**が生成されます）
2. 必要に応じてプロンプトを編集
3. 「🎨 画像を生成」をクリック（**枠・文字は自動で統一描画**されます）
4. 少し待つと画像が表示されます（10〜30秒程度）

### ステップ4: ダウンロード

生成された画像は「💾 ダウンロード」ボタンで保存できます。

## 各タブの詳しい使い方

### 📚 テンプレート

一番かんたんな方法です。

1. **カテゴリを選択**: 基本、分類別、性質、消火、法令から選ぶ
2. **テンプレートをクリック**: 好きなカードをクリックするとプロンプトがセットされる
3. **必要なら編集**: 中央イラストの指示を編集
4. **画像を生成**: ボタンをクリック

**カテゴリ一覧**:
- **基本**: カード裏面のデザイン、フレームなど
- **分類別**: ガソリン、灯油など物質別のカード
- **性質**: 引火点、比重などを強調したカード
- **消火**: 消火方法を表示したカード
- **法令**: 指定数量や貯蔵基準のカード

### 🧪 物質情報

物質のデータを入力して、カードを作ります。

1. **サンプルをロード**（おすすめ）: ボタンをクリックするとデータが自動入力される
2. **または手動入力**: 各項目を自分で入力
   - 物質名（必須）
   - 分類
   - 引火点、発火点、沸点
   - 比重、蒸気比重
   - 水溶性
   - 指定数量
   - 特記事項
3. **プロンプトを生成**: ボタンをクリック
4. **画像を生成**: ボタンをクリック

**サンプル物質**:
- ジエチルエーテル（特殊引火物）
- 二硫化炭素（特殊引火物）
- ガソリン（第1石油類）
- アセトン（第1石油類・水溶性）
- エタノール（アルコール類）
- 灯油（第2石油類）
- 軽油（第2石油類）
- 重油（第3石油類）

### 🧯 消火方法

消火方法に特化したカードを作ります。

1. **サンプルをロード**: 消火栓、スプリンクラーなどから選ぶ
2. **または手動入力**:
   - 消火器具名（必須）
   - 種別
   - 第4類危険物に使用可否
   - カードの効果
   - 事実根拠
3. **プロンプトを生成** → **画像を生成**

### 🚚 運搬

運搬に関するカードを作ります。

1. **サンプルをロード**: 密閉・遮光被覆などから選ぶ
2. **または手動入力**:
   - カード名（必須）
   - 効果内容（強化ポイント）
   - 事実根拠
3. **プロンプトを生成** → **画像を生成**

### 📜 法令

法令に関するカードを作ります。

1. **サンプルをロード**: 使用停止令、改修命令などから選ぶ
2. **または手動入力**:
   - カード名（必須）
   - 妨害効果
   - 法的根拠
3. **プロンプトを生成** → **画像を生成**

### 📦 一括発行

CSVから複数カードをまとめて作成します。

1. **テンプレートCSVをダウンロード**
2. **生徒作成データをCSVでアップロード**
3. 読み込み時に以下を自動実行
   - 効果テキストの文面整頓
   - 危険物取扱者試験向けの事実補足
4. プレビューで「原文効果 / 整頓後 / 学習補足」を確認
5. 「確認しました」にチェックして一括発行
6. 個別保存または全件ダウンロード

### ✏️ 直接入力

自分でプロンプトを自由に書きます。

**プロンプトの基本構造（中央イラストのみ）**:
```
中央イラスト：○○、主題、色調、雰囲気、背景の指示
```

**例**:
```
燃料缶とオレンジの炎を主題にしたイラスト。オレンジ系の色調、強い炎の光、背景は控えめで主題を邪魔しない。カードゲーム用の高精細イラスト。
```

## よくある質問（FAQ）

### Q: 画像が生成されない

**A**: 以下を確認してください：
1. APIキーが正しく入力されているか
2. プロンプトが空になっていないか
3. インターネット接続があるか

### Q: 「API利用制限に達しました」と表示される

**A**: APIの利用回数制限に達しました。少し時間を置いて（数分〜数十分）からもう一度試してください。

### Q: 「安全上の理由で画像を生成できませんでした」と表示される

**A**: AIが不適切と判断した内容は生成されません。プロンプトの内容を変えてみてください。

### Q: 思ったような画像にならない

**A**: プロンプトを工夫してみてください：
- より具体的な説明を追加する
- 「カードゲーム風」「イラスト調」などのスタイル指定を追加
- 色や配置を詳しく指定する

### Q: ダウンロードした画像が小さい

**A**: 現在はカード画像が **1000 x 1500px** で生成されます。印刷用には十分なサイズです。

## 作ったカードの活用方法

1. **印刷してカードゲームに**: 厚紙に印刷して切り取る
2. **暗記用**: スマホに保存して通学中に復習
3. **グループ学習**: みんなで違うカードを作って見せ合う
4. **テスト対策**: 苦手な物質のカードを重点的に作る
